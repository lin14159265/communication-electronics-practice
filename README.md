# 通信电子线路刷题系统

独立静态网页，包含通信电子线路选择题、多选题、填空题题库。

## 题库范围

- 单选题：152 题，题号 1-152
- 多选题：5 题，题号 153-157
- 填空题：49 题，题号 158-206
- 合计：206 题

## 功能

- 顺序练习、随机练习
- 错题重练、未掌握题、收藏题、不确定/待复习
- 单选、多选、填空即时判题
- 批量判题模式
- 做题统计、错题统计
- 本地 localStorage 保存学习记录
- 学习记录导入/导出
- PWA 支持，部署到 HTTPS 后可添加到主屏幕

## 使用方式

直接打开 `index.html` 即可使用。若要启用 PWA 离线缓存，请部署到 GitHub Pages、Netlify、Vercel 等 HTTPS 静态网页环境。

## 边界

本项目没有导入原单片机题库，也没有依赖原单片机刷题网站。旧网站只作为交互逻辑参考。
